- Coursera 요약
    - In the last video, we worked through an example of using a computation graph to compute a function J.
    - This video takes a cleaned up version of that computation graph and shows how to use it to figure out derivative calculations for that function J.
    - To compute the derivative of J with respect to v, we can use the equation J= 3v.
    - This is analogous to the example from the previous video, where f(a) = 3a and af/da = 3.
    - To compute the derivative of J with respect to a, we can use the equation J = 3V, where v=a+u.
    - If we increase a by 0.001, then v will increase by 0.001 and J will increase by 0.003.
    - This means that the derivative of J with respect to a is equal to 3.
    - This is called one step of backpropagation.
    - In code, the derivative of the final output variable with respect to any other variable is represented by the variable name dvar.
    - Course lesson is on Derivatives with a Computation Graph
    - Backpropagation is partially done through the computation graph
    - dv = 3 and da = 3
    - To compute dJ/du, start with u = 6 and bump up u to 6.001, then v goes up to 11.001 and J goes from 33 to 33.003
    - Increase in J is 3x, so dJ/du= 3x1=3
    - To compute dJ/db, use the chain rule for calculus and dJ/du x du/db = 6
    - To compute dJ/dc, dJ/dux du/dc = 9
    - Most efficient way to compute derivatives is right to left following the direction of the red arrows

### Computing derivatives

![](/assets/a5984a54-e437-4439-bfab-08a88f27c6b9.png)

- 미분 과정
    - J = 3v
    - v = a + u
    - a = 5, u = bc
    - b = 3, c = 2
    - 오른쪽으로 왼쪽 방향으로 (역전파)
- J = 3v → v = a + u
    - v에 따른 J의 변화량 (코드에서 ”dv” 라고 나타냄)
    
    $$
    \frac{dJ}{dv} = 3
    $$
    
- v = a + u → a = 5
    - a에 따른 v의 변화량
    
    $$
    \frac{dv}{da}=1
    $$
    
    - a에 따른 J의 변화량
        - 미분의 체인룰 이용
        - 코드 상에서 “da” 라고 나타냄
    
    $$
    \frac{dJ}{da}=\frac{dJ}{dv}\frac{dv}{da}=3
    $$
    
- v = a + u → u = bc
    - u에 따른 v의 변화량
    
    $$
    \frac{dv}{du} = 1
    $$
    
    - u에 따른 J의 변화량
        - 미분의 체인룰 이용
        - 코드 상에서 “du” 라고 나타냄
    
    $$
    \frac{dJ}{du}=\frac{dJ}{dv}\frac{dv}{du}=3
    $$
    
- u = bc → b = 3
    - b에 따른 u의 변화량
    
    $$
    \frac{du}{db} = 2
    $$
    
    - b에 따른 J의 변화량
        - 미분의 체인룰 이용
        - 코드 상에서 “db” 라고 나타냄
    
    $$
    \frac{dJ}{db}=\frac{dJ}{du}\frac{du}{db}=6
    $$
    
- u = bc → c = 2
    - c에 따른 u의 변화량
    
    $$
    \frac{du}{dc} = 3
    $$
    
    - c에 따른 J의 변화량
        - 미분의 체인룰 이용
        - 코드 상에서 “dc” 라고 나타냄
    
    $$
    \frac{dJ}{dc}=\frac{dJ}{du}\frac{du}{dc}=9
    $$
    
- 코드 상에서 최종 출력 변수를 “dvar”로 나타냄
    - 위에 적었듯이 “du”, “dv”, … 같이 변수명을 적음
