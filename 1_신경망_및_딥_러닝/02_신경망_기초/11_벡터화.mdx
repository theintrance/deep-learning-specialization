### Vectorization

데이터를 적절히 행렬(벡터) 화 하여 명시적 for/while loop 를 사용하지 않고 계산을 빠르게 수행하도록 하는 기술

로지스틱 회귀의 과정 중 아래를 계산할 때,

$$
z=w^Tx+b
$$

$w$와 $x$는 각각 $m$차원 행 벡터이기 때문에 python 에서 이를 구현하면 아래와 같이 할 수 있음

```python
# w, x, m, b are defined

z = 0
for i in range(m):
		z += w[i] * x[i]
z += b
```

하지만 위 python 코드는 매우 느림

numpy 에서는 내부적으로 SIMD 를 사용한 최적화가 되어있는 vector 연산을 제공함

```python
import numpy as np

# w, x, b are defined
z = np.dot(w, x) + b
```
