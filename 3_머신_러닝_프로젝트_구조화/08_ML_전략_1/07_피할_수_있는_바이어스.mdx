---
title: 피할 수 있는 바이어스
---
| Humans | 1% |
| --- | --- |
| Training Error | 8% |
| Dev Error | 10% |
- 위와 같은 경우, 인간 수준과 훈련 세트 에러의 차가 크기 때문에 모델을 개선해야하고, 바이어스를 줄이는 것에 집중해야 함

| Humans | 7.5% |
| --- | --- |
| Training Error | 8% |
| Dev Error | 10% |
- 만약 데이터에 흐리거나 모호하여 인간도 7.5% 수준의 에러일 경우, 훈련 오류는 괜찮은 수준이라고 판단하고, 분산을 줄이는 것에 집중해야 함

- 즉 Human-level error 을 Bayes Optimal Error 의 프록시 혹은 추정치로 생각한다. 
(인간 수준이 모델 평가의 기준이 된다.)
- 인간 수준 에러와 훈련 에러의 차를 “피할 수 있는 바이어스” (Avoidable bias) 라고 한다.
    - 개선의 여지가 있으며 개선이 필요하다는 뜻.
    - 두번째 예시의 경우 회피 가능 편향은 0.5%이다.
    - 두번째 예시의 경우 분산은 2% 이므로, 2% 를 줄이는 것이 더 쉽고 이를 줄이는 것에 집중하는 전략을 택한다.
- 만약 훈련 에러가 베이즈 에러보다 낮은 경우 모델이 과적합 된것이다.
