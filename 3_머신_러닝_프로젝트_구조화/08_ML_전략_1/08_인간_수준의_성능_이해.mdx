### 인간 수준 에러와 바이즈 에러

- 만약 X-ray 사진을 바탕으로 골절 유무를 판단하는 모델을 만든다면, 아래와 같이 사람에 따라 에러값이 다를 것이다
    - 일반 사람: 3% error
    - 일반 의사: 1% error
    - 전문 의사: 0.7% error
    - 전문의 팀: 0.5% error
- 이렇게 평가하는 사람에 따라 다른데, 인간 수준의 오류를 어떻게 정의하나?
- Andrew Ng은 해당 경우 0.5% 를 베이즈 에러로 간주한다.
    - 최대한 오류가 낮아야 하고 일반 의사보다는 모델의 성능이 높아야 하기 때문
- 위 예시처럼 모델의 목적에 따라 인간 수준의 레벨을 어느정도로 설정하고 모델을 평가할 기준을 세울건지에 대한 검토가 필요하다.
- 베이즈 오류를 어떻게 설정하는지에 따라 모델의 개선 방향을 설정하기 때문에 이는 매우 중요함
