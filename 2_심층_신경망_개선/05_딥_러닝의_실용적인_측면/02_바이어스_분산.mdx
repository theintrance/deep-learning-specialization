### 바이어스와 분산

- 편향(bias)이 크다 → 훈련한 모델의 정확도가 낮다. (underfitting)
- 분산(variance)이 크다 → 모델이 훈련 세트에 과도하게 적합되어있다. (overfitting)

| **상황** | **Train Set Error** | **Dev Set Error** | **설명** |
| --- | --- | --- | --- |
| 과적합 (Overfitting) | 낮음 | 높음 | 모델이 훈련 데이터에 너무 맞춰져 있어 새로운 데이터에 일반화되지 않음 |
| 과소적합 (Underfitting) | 높음 | 높음 | 모델이 훈련 데이터에도 충분히 맞지 않음 |
| 잘 학습됨 (Well-fitting) | 낮음 | 낮음 | 모델이 훈련 데이터와 검증 데이터 모두에서 좋은 성능을 보임 |
